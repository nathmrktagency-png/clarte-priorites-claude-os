## Positionnement business

Clarté & Priorités Marketing est un cabinet junior structuré de cadrage marketing pour dirigeants de PME locales B2C.

Le cabinet vend :
- clarté ;
- diagnostic ;
- priorisation ;
- méthode ;
- cohérence ;
- pilotage ;
- livrables professionnels.

Le cabinet ne vend pas :
- ROI garanti ;
- croissance garantie ;
- ventes garanties ;
- agence 360 degrés ;
- exécution directe sans audit.

Workflow :
Qualification → Audit payant → Dispositif Simple ou Structuré → Accompagnement éventuel.

## Règles de modification

Avant toute modification importante :
1. analyser la demande ;
2. proposer un plan ;
3. attendre validation si la modification est structurante ;
4. modifier uniquement les fichiers nécessaires ;
5. résumer les changements ;
6. signaler les risques.

Ne jamais :
- supprimer sans validation ;
- renommer massivement sans validation ;
- déplacer des dossiers sans validation ;
- ajouter de données sensibles ;
- inventer une règle business ;
- modifier le positionnement ;
- ajouter une promesse de ROI.

## Style attendu

Français clair, sobre, structuré et pédagogique.
