# Prompts

Bibliothèque de prompts réutilisables pour Clarté & Priorités Marketing.

Exemples :
- prospection ;
- qualification ;
- audit ;
- priorisation ;
- contrôle qualité ;
- branding ;
- KPI ;
- juridique.

Tous les prompts doivent respecter :
- aucune promesse de ROI ;
- aucun discours agence 360 ;
- “À vérifier” si information manquante ;
- validation humaine avant action sensible.
