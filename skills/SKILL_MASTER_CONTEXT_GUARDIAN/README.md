# SKILL_MASTER_CONTEXT_GUARDIAN

## Objectif

Cette Skill aide Claude à vérifier qu’une réponse, un livrable, une offre, un message ou une recommandation respecte le Master Context de Clarté & Priorités Marketing.

## Rôle

Elle agit comme garde-fou contre :
- les promesses de ROI ;
- les garanties de ventes ;
- le positionnement agence 360 ;
- les dispositifs proposés sans audit ;
- les données inventées ;
- les demandes hors-scope ;
- les risques de confidentialité ;
- les contradictions documentaires.

## Règle

Cette Skill ne remplace pas le Master Context.
Elle rappelle à Claude comment appliquer les règles du Master Context.

## Données sensibles

Cette Skill ne doit contenir aucune donnée client réelle.
